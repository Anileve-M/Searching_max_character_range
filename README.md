# Поиск максимального интервала символов

## Описание
Программа решает задачу поиска максимального интервала, состоящего только из символов 'a', в наборе случайно сгенерированных строк. Программа представлена в двух вариантах реализации с использованием многопоточности.


## Технологический стек
1. Thread/Runnable
2. Callable/Future
3. ExecutorService
4. Java

## Архитектура приложения
## Класс `Max` (реализация с `ThreadPool`)
### Основные компоненты:
### 1. Генерация данных:
- Создается 25 строк длиной 30,000 символов
- Символы выбираются из набора "aab" (66% 'a', 33% 'b')

### 2. Многопоточная обработка:
- Создается пул потоков размером 25 (по одному на каждую задачу)
- Для каждой строки создается Callable<Integer> задача
- Результаты собираются в список Future<Integer>

### 3. Алгоритм поиска:
- Тройной вложенный цикл для проверки всех возможных подстрок
- Сложность O(n³) для каждой строки

### 4. Ключевые методы:
 - main(String[] args) - точка входа, координация всего процесса
- generateText(String letters, int length) - генератор случайных строк

### 5. Особенности реализации:
- Использует ExecutorService для управления потоками
- Возвращает единственное значение - максимальный интервал среди всех строк

Автоматическое ожидание завершения всех задач.


## Класс `Main` (реализация с `Thread`)
### Основные компоненты:
### 1. Управление потоками:
- Создается список Thread объектов
- Каждый поток обрабатывает одну строку независимо
- Используется join() для ожидания завершения всех потоков

### 2. Вывод результатов:
- Для каждой строки выводится префикс (первые 100 символов) и найденный максимальный интервал
- Результаты выводятся по мере завершения потоков

### 3. Особенности реализации:
- Прямое создание и управление потоками
- Параллельный вывод результатов
- Более простая архитектура


## Выходные данные
### Класс `Max` выводит:
***Max size***: X - максимальный найденный интервал среди всех строк

***Time***: Yms - общее время выполнения

### Класс `Main` выводит:
***Для каждой строки***: первые 100 символов и найденный максимальный интервал

***Time***: Yms - общее время выполнения
